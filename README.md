# eas-sdk

一个用于提升校园教务系统（EAS）使用体验的轻量级 SDK。  
目标是提供更稳定、可扩展、可自动化的信息获取与通知能力。

---

## 项目背景

当前教务系统在使用过程中存在以下问题：

- 偶发性访问不稳定  
- 交互体验较差  
- 缺乏实时通知能力  
- 不支持个性化订阅  

本项目旨在构建一个用户侧增强层，用于信息整理与推送。

---

## 功能规划

围绕教务数据的读取与通知：

- 课程信息获取  
- 考试安排解析  
- 成绩变动检测  
- 选课状态监控  
- 消息推送系统  

---

## Bot 接入计划

计划支持：

- 基于 NapCat 的 QQ Bot 推送  
- 事件驱动通知  
- 自定义订阅规则  

当前阶段仅考虑 NapCat 实现。

---

## 设计原则

- 默认不存储账号凭据  
- 本地化配置  
- 模块化架构设计  
- 尽量降低对原系统的侵入性
